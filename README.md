# cn-ssl-notes

## 项目简介

本项目是一个以 HTML 页面形式归档和发布各类 SSL/TLS 相关笔记的仓库。页面内容独立，不针对任何具体域名或网站，旨在以静态文档的方式整理和分享与 SSL/TLS 协议、证书、配置、常见问题等相关的技术记录与参考信息。

## 目录说明

```
cn-ssl-notes/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（如 CSS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立的笔记页面，文件名简洁明了，内容自包含。
- `assets/`：存放页面所需的公共资源文件，便于统一管理和引用。

## 页面归档说明

- 所有页面均采用标准 HTML 编写，可直接在浏览器中打开查看。
- 页面内容聚焦于 SSL/TLS 相关主题的技术笔记，不涉及任何商业推广或特定网站关联。
- 归档内容可能随仓库更新而增加或修订，请以最新版本为准。

## 维护说明

- 本仓库由个人维护，不定期更新页面内容。
- 欢迎通过 Issue 提出建议或报告问题，但请注意不要包含具体域名、网址或营销信息。
- 仓库不提供任何形式的保证或承诺，内容仅供参考学习使用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的任意 HTML 文件即可查看对应笔记。
3. 也可直接通过 GitHub Pages 等静态托管服务浏览页面（如有配置）。
